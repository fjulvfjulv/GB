## Списки

Списки ненумерованные можно создавать с помощью звездочек (*) или знаком + 
Пример:
* Элемент 1
* Элемент 2
+ Элемент 3

Списки нумерованные можно создавать с помощью цифр.
Пример: 
1. Элемент 1
2. Элемент 2
